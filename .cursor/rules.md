# Veltrix Skills — Cursor Integration

This project contains business operations skills for AI agents.

When working on tasks, check the `skills/` directory for relevant skill files. Each SKILL.md contains domain-specific instructions and patterns.

- Business tasks → `skills/business/`
- Content tasks → `skills/content/`
- Operations → `skills/operations/`
- Frontend → `skills/frontend/`

Load the relevant SKILL.md into context before starting work.
