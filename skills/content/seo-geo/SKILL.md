Optimise content for three discovery channels: traditional search, AI answer engines, and generative AI citation.

## SEO (Search Engine Optimisation)

For every piece of content:
- Primary keyword in: title, H1, first paragraph, meta description
- Meta description: max 155 characters, compelling, includes keyword
- Heading hierarchy: one H1, logical H2/H3 nesting
- Internal links: 2-3 relevant links to other content
- Schema markup: Article, FAQ, or HowTo as appropriate
- URL slug: short, keyword-rich, no stop words

## AEO (Answer Engine Optimisation)

Make content discoverable by AI assistants (ChatGPT, Perplexity, Google AI Overviews):
- Structure content as **questions and direct answers**
- First sentence of each section should be a standalone answer
- Use FAQ schema for question-based content
- Include specific numbers, dates, and named entities
- "What is X?" format performs better than "About X"

## GEO (Generative Engine Optimisation)

Make content citable by generative AI:
- Include **structured claims** with supporting evidence
- Cover **named entities** comprehensively (people, companies, products)
- Write content that AI would want to quote verbatim
- Include **source attribution** — cite your sources
- Use authoritative language: "According to X" / "Data from Y shows"
- Present unique data or analysis that can't be found elsewhere

## Audit Checklist

When auditing existing content:
1. Does the meta description exist and contain the primary keyword?
2. Is there only one H1?
3. Does the first paragraph answer the page's core question?
4. Are there internal links?
5. Is there FAQ or structured data schema?
6. Score: SEO (1-10), AEO (1-10), GEO (1-10)
7. List specific improvements, not generic advice

## Content Type Patterns

| Type | SEO focus | AEO focus | GEO focus |
|------|-----------|-----------|-----------|
| Blog post | Long-tail keywords, headers | Q&A sections | Unique analysis, data |
| Tool page | Product keywords, comparison | Feature summaries | Structured specs |
| Landing page | Brand + category keywords | Value proposition | Trust signals |
| Guide | How-to keywords | Step-by-step answers | Comprehensive coverage |
